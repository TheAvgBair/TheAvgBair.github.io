Level.io Support

Thanks for using Level.io. If you run into a crash, broken save, confusing quest reward, UI issue, or anything that feels off, send a message and we will take a look.

Contact

Email support at:

TheAvgDevs@gmail.com

What To Include

Please include as much of this as you can:

  • App version and build number.
  • iPhone model and iOS version.
  • What you were trying to do.
  • What happened.
  • What you expected to happen.
  • Whether Reminders sync, Cloud backup, notifications, or diagnostics were enabled.

In-App Diagnostics

If diagnostics sharing is enabled in Settings, Level.io can prepare a local diagnostic report. Please review the report before sending it. Diagnostics sharing is optional and manually controlled.
